
## Install

``` bash
pipenv install
pipenv run python manage.py migrate
npm install
```

## Initialize data

``` bash
python manage.py loaddata occupations
```

## Start project

```
## Django dev server
pipenv run manage.py runserver

## Npm dev server
npm run dev
```
